# Code of Conduct

This project uses a simple standard: be direct, kind, and useful.

Disagreement is welcome. Personal attacks, harassment, slurs, or repeated bad faith behavior are not.

Assume benchmark results can be messy and gold labels can be wrong. Critique claims, gold labels, cases, and scores with enough detail that someone else can verify or improve them.

Maintainers may remove comments, issues, pull requests, or participation that make the project harder for reasonable people to work on.
